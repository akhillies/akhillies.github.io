Bezier Curves

So this project is completely different from the others in that not only did I start from scratch, but I was learning a concept I had never touched before - at least not with the name Bezier Curves. By using Adobe Illustrator, I knew about curve representations as 4 control points, but what I never thought of was using 4 of those curves to represent a surface, which are otherwise known as Bezier Surfaces. This project was one of those things where you needed one simple, key understanding, and everything would fall into place (especially since this project was geared towards data structure side of graphics rather than graphics concepts themselves). I would have to say that this project was definitely easier than the Ray Tracer considering I did not have to directly deal with reflections and transformations (I used OpenGL this time). In the end, this project was probably the one I had the most fun with because of all the user interactions I added into the project. Finishing this project helped me realize that I want to get into modeling and character design for animations (still haven't decided between movies or video games). The README file has all the features I implemented as well as the controls/commands for the program. To learn more about this project, you can email me at akhilbatra@berkeley.edu

.

Personal Comments/Feedback

So this project started as being one of the harder ones - mainly because while I understood the theory and the applications of Bezier curves, I did not understand how they were used to physically represent the object and therefore how to make a program that took surfaces and made them into objects. However, because of a very good high school friend of mine who did understand Bezier surfaces, I got the one small, vital piece of information needed for me the understand how Bezier surfaces were coded and I quickly finished the project after that. From then I just worked on adding extra features - mainly focused on adding user interactions features like selecting of objects, moving individual objects as well as the whole scene, and even changing the type of display the object had (like wire frame, smooth or flat shading, etc.). Overall, I had the most fun with this project, nothing was frustrating at all, which kind of made me realize how much I like modeling and making poses or still scenes of objects and people.
